### 技术栈说明

- 前端展示系统预计使用最新的 Vue3,完成博客的展示及其他信息
- 管理后台使用 react hooks 编码,对比 hooks 和 Vue3 的 composition 优劣
- 后端 使用阿里框架 eggs 作为博客数据后台服务
- 使用 koa2 作为管理后台服务,装饰器路由
- 所有项目使用 typescript,践行函数式编程思想
